> *"Sebuah perjalanan pulang ke dalam diri."*

**Resonansi** adalah sebuah web eksperimental yang menggabungkan seni sastra (puisi) dengan desain web interaktif. Proyek ini dirancang sebagai ruang aman (*safe space*) digital untuk merenung, memvalidasi perasaan, dan menemukan kembali ketenangan setelah masa-masa sulit.

Dibangun dengan estetika minimalis bergaya novel, web ini mengajak pengguna menelusuri bab demi bab perjalanan emosional dari kegelapan menuju cahaya.

## âœ¨ Fitur Utama

* **Atmospheric Scroll Experience:** Menggunakan *Scroll Snap* untuk pengalaman membaca yang fokus dan terarah per bab.
* **Dynamic Theme Transition:** Transisi warna latar belakang yang halus dari gelap (Dark Mode) ke terang (Light Mode) seiring perjalanan cerita, menyimbolkan proses penyembuhan.
* **Interactive Particles:** Efek partikel canvas yang tenang dan responsif terhadap tema warna.
* **Minimalist Typography:** Menggunakan font *Cormorant Garamond* untuk nuansa elegan dan menenangkan.
* **Responsive Design:** Tampilan yang optimal baik di Desktop maupun Mobile (Smartphone).
* **Animation:** Efek *fade-in* yang lembut saat elemen masuk ke layar.
